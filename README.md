# test
2D 物理エンジン
    必要な環境
        python3, pygame
    実行方法
        python3でmain.pyを実行
    操作方法
        入力
            左クリック
                ボールをドラッグ
            右クリック
                ボールを削除
            スペース
                ボールを追加
            t
                一時停止切り替え
        メニューバー
            Change Gravity
                x方向・y方向の加速度を入力
